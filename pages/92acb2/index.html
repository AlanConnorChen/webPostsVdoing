红外光学定位与数字化种植牙手术导航系统研发

# 项目描述

企业委托横向项目。团队辅助企业全面梳理相关软件代码，明确出其中的开发逻辑，使其具备二次开发能力。此外，团队开发出了可适配企业相关系统的光学定位追踪主机，在假体实验时种植体平均颈部偏差小于1mm，平均根尖端误差小于1.5mm，平均角度偏差小于5度。

# 硬件选型

# 相机

团队采用的是基于红外光学成像的光学导航系统，选取黑白单色相机以及滤光片的组合来捕获器械上的特征闪光点。具体的相机设备名称为Basler ace acA1440-220um，采用USB3.0接口与计算机连接。 相机默认的图像采集分辨率为1440×1080，但该分辨率可以在官方提供的SDK工具中进行修改，使用的传感器类型为Sony IMX273LLR-C。相机拍摄x轴坐标范围是450~550mm，y轴坐标范围是350-450mm，z轴坐标范围是630~1000mm。据此，团队设置实验平台相机的水平视场角为63.7度，而垂直视场角为50度，设置焦距为4mm，视场范围三维可视化如下图。 Basler ace acA1440-220um相机成像范围 相机镜头组合

# 光源

成像平台所需要的光波长在850nm~900nm范围内，选取的滤光片对780nm~910nm范围的光谱响应灵敏度最高，对其他波长范围的光则会将其过滤，不予其通过滤光片，实现排除其他环境光干扰的作用。 红外滤光片透光范围曲线 成像平台红外光源灯板设计

# 红外光学成像平台

团队严格依据相关相机型号与感光范围设定遴选相关成像硬件，构建近红外光成像平台及其相关电路实现，平台实物图如下所示 近红外光学成像导航平台实物 右目相机 左目相机

# 图像处理

二值化。在执行特征匹配之前，需要对图像进行预处理操作。首先，团队将左右目图像转化为灰度图，这一步操作将原始图像维度从3×1440×1080转化为1×1440×1080，由于采集到的图像本身即为黑白单色，其视觉表达信息和原图相同。接着，团队通过设置阈值对图像进行二值化处理，将图像中的像素值根据预先设定的阈值分为两类：高于阈值的像素被设置为白色；低于阈值的像素被设置为黑色。

右目图像二值化处理

轮廓检测。团队将二值化图像作为输入，随后在二值化图像中查找轮廓信息。使用OpenCV中cv::findContours检索最外层轮廓，并且使用简单的链码近似方法来压缩水平、垂直和对角线方向的冗余点，从而只保留轮廓的关键点，这一步操作将会返回一个包括轮廓点集合的向量信息。

右目图像特征点轮廓检测

点群分类。对双目相机拍摄到的图像标识点进行K-means点群聚类算法，使用K-means++算法初始化点群中心，同时将点群聚类数量设置为2。将左右图中的植牙反射环上的光点作为一个点簇，植牙定位器上的光点作为另一个点簇。
