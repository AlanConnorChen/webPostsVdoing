        vuepress-theme-vdoing配置Google Analytics

        # 获取Google Analytics衡量ID

        谷歌访问统计地址Google Analytics (opens new window),若是没有账号，注册登录即可，谷歌浏览器可直接使用已登录的谷歌账号进行登录。按照网站提示信息创建数据流，其中数据流名称自定义即可，网址填写博客所在的网页地址，得到G-开头的衡量ID，如下图。

        # Vuepress插件配置

        终端切换到网页项目目录

        cd /home/alan/Web/vuepress-theme-vdoing
        1

        使用npm安装插件VuePress Google Analytics 4 Plugin (opens new window),该插件支持基于VuePress v1.x版本开发的网页

        npm install vuepress-plugin-google-analytics-4
        1

        随后在config.ts插件配置中添加如下代码，gtag为Google Analytics数据流统计的衡量ID

            [
	    'google-analytics-4',
	    {
	      gtag: 'G-NYHC43LY5Z'
	    }
	  ],
        1

        # 网站数据概览

        部署完成网站后，回到Google Analytics网站即可查看到网站报告概况、实时概览情况，此时对前面配置网页的点击访问操作均可被记录。
