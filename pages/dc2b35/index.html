        vuepress-theme-vdoing嵌入YouTube视频

        # 来由

        Vdoing主题不提供视频嵌入相关功能，仓库的往年issue也提到了类似问题，仓库维护者给出的vuepress-plugin-smplayer插件已经不再维护，也无法找到相关的文档。

        Vdoing Github Issue

        # vuepress-theme-hope

        于是转为使用vuepress-theme-hope (opens new window)主题进行设置，但一番调试后发现右侧作者信息栏总是无法隐去，在美观度上个人并不喜欢。 后续也查找了VidStack (opens new window)、Artplayer (opens new window),但也是没调整成功。

        # 嵌入YouTube视频

        这里其实是AI给出的解决方案，想在Vdoing主题基础上调通Artplayer，但是无法调通，看到了AI生成的后面答案。 使用iframe结合布局调整即可，还需注意，嵌入YouTube视频的时src给的不是视频链接，而是https://www.youtube.com/embed/VIDEO_ID，参考freeCodeCamp论坛 (opens new window)，代码如下。

          <div style="display: flex; flex-wrap: wrap; justify-content: center;">  
    <div style="position: relative; width: 100%; max-width: 420px; height: 0; padding-bottom: 56.25%; margin: 5px;">  
        <iframe  
            src="https://www.youtube.com/embed/hFqYem2mwKA"  
            frameborder="0"  
            allowfullscreen  
            style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;">  
        </iframe>  
    </div>  

    <div style="position: relative; width: 100%; max-width: 420px; height: 0; padding-bottom: 56.25%; margin: 5px;">

        </iframe>  
    </div>
